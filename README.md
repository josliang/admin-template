# Admin Template

vue3 管理后台方案，具有动态路由、权限管理，轻量简洁。

## 项目架构

- 💪 Vue 3、Vue-Router 4、Vite、esbuild
- ☀️ Pinia 状态管理、Uno原子化Css
- 🎨 Naive 组件、Icons 自动按需导入
- ⚙️ vue 组件、hooks 自动导入

## 运行命令

### 安装依赖

```sh
yarn
```

### 项目运行

```sh
yarn dev
```

### 项目打包

```sh
yarn build
```
